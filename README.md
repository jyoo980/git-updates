# git-updates

Git Updates is a lightweight GitHub client developed for iOS. I spend a lot of time on GitHub, and thought it'd be a fun exercise for me to develop a mobile app with some of the basic functionalities of GitHub, while providing a bit of a social flavour. Here are some screenshots of the app below:

</p>

## Technologies
* <strong>GitHub API</strong>: documentation can be found [here](https://developer.github.com/v3/)
* <strong>Caching</strong>: `FollowedUserCache.swift`

## Development dependencies
* <strong>Cocoapods</strong>: install Cocoapods globally, `sudo gem install cocoapods`.
* <strong>Alamofire/AlamofireImage</strong>: navigate to project directory, run `pod init`, open `Podfile` in the text editor of your choice, include the line: `pod 'AlamofireImage', '~> 3.3'` under the line `target`.
